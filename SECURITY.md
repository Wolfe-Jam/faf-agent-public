# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in `faf-agent` (the source code, the spec, or any artifact in this About Repo), please report it privately:

**Email:** team@faf.one

We aim to acknowledge within 48 hours. Please do not file public GitHub issues for security concerns.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Scope

This About Repo carries public artifacts (README, spec, project.faf). The agent's source code lives privately at `Wolfe-Jam/faf-agent`. Security reports for the source code should also go to `team@faf.one`.
